# Crush It Server

This is the backend of the Crush It application.

## Development

1. **Important!**

- When working in the backend, you need to make sure you are in `app/api`

2. **Install/Update Dependencies:**
   ```bash
   yarn install
   ```
3. **Run the Server:**

- For development:
  ```
  yarn dev
  ```

4. **Access the API:**
   http://localhost:443
